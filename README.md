# Flutter Auto Localizer (VS Code Extension)

This extension automates the localization process for Flutter apps. It extracts hard-coded strings from your Dart code and generates the corresponding ARB files automatically.

## Features (Implemented so far)
- [x] Extracts string literals from Dart files.
- [x] Ignores imports and comments.
- [x] Generates generic keys (camelCase) for strings.
- [x] Creates and updates `lib/l10n/app_en.arb` file automatically.

## How to Run (Dev Mode)
1. Clone the repository.
2. Run `npm install`.
3. Press `F5` to launch the Extension Development Host.
4. Run command: `Flutter Auto Localizer: Start`.
